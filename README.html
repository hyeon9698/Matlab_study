고전적인 도형 그리기 방식

circle

1. 수식

x2+y2=r2x^2 + y^2 = r^2x2+y2=r2

x2=r2−y2x^2 = r^2 - y^2x2=r2−y2

x=±r2−y2x = \pm \sqrt{r^2-y^2}x=±r2−y2​

x1=−r2−y2x_1 = -\sqrt{r^2 - y^2}x1​=−r2−y2​

x2=+r2−y2x_2 = +\sqrt{r^2 - y^2}x2​=+r2−y2​

2. 매틀랩 코드

clear
clc

i = 5000

A = zeros(i);
r = 500;

for y = -r:r
    x1 = -sqrt(r.*r-y.*y);
    x2 = sqrt(r.*r-y.*y);
    
    for x = x1:x2
        A(round(x+i/2),round(y+i/2)) = 256;
    end
end

figure(3)
image(A)
axis ('square')
grid

Vignetting Effect 만들기

1. 가우시안 함수

gaussian=e−α(x2+y2)gaussian = e^{-\alpha(x^2+y^2)}gaussian=e−α(x2+y2)

α=weight\alpha = weightα=weight

2. 매틀랩 코드

A = imread('image.jpg')
%image를 불러들여 A에 저장

[size_x size_y size_z] = size(A);
%image의 X,Y,채널 값을 저장 한다.

y = linspace(-1,1,size_x);
x = linspace(-1,1,size_y);

%meshgrid 생성

%image RGB중 R추출

%가우시안 필터 가중치 = 3

A_R1 = A_R([1:round(size_x/2)],[1:size_y/2]);
